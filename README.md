# Wallpaperz

A wallpapers app using AngularJS and Node.js

## Front-End:
- AngularJS
- Bootstrap
- Font-Awesome
- Wallgig API (https://www.mashape.com/jianyuan/wallgig-v1)
- Underscore
- ngInfiniteScroll (https://sroze.github.io/ngInfiniteScroll/)

## Back-End:
- Node.js
- Express.js
- MongoDB and Mongoose

## How to Run:
- npm install
- Start mongod
- npm start
- Visit localhost:8080
